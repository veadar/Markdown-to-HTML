#Markdown to HTML

##これは何？

簡素なMarkdownプレビューアです。

サービスメニューもしくはドラッグ＆ドロップからMarkdownファイルを開きます。

##対応 OS

OS X 10.10以降(おそらく)

##スクリーンショット

![スクリーンショット](./screenshot1.png)

![スクリーンショット](./screenshot2.png)

##機能

- タイトルバーとフチなしウィンドウ
- クリックスルーかつ最前面
- HTMLをクリップボードに格納

Downloadをクリック → [Download](https://github.com/veadar/Markdown-to-HTML/releases)
